เอสเอสยูพี กรุ๊ป (SSUP Group)

งานเกี่ยวกับ SAP:

  • พัฒนา SDAL ระบบประมวลหาความต้องการวัตถุดิษ เพื่อที่ช่วยตัดสินใจในการสั่งซ์้อ PO ใน SAP โดยผ่าน API
  • พัฒนา Check Short ระบบที่หาวัตถุดิษที่ขาดหายก่อนการผลิต
  • พัฒนา API ในส่วนเชื่อมต่อ SAP B1 ต่างๆ เช่นการเบิกของ การโอนของ การปรับปรุงข้อมูลต่างๆ เป็นต้น
  • ออกรายงานต่างๆ SAP B1
